# Edit Checklist

Use this checklist before returning polished Korean text.

## Meaning

- Same intent as the source.
- No new facts.
- No removed constraints.

## Tone

- Matches the user's requested tone.
- Does not become overly formal by accident.
- Uses consistent honorific level.

## Language

- No translated sentence order.
- No repeated nouns or filler.
- No awkward loanword stacking unless the term is standard.
- Technical terms remain intact when they are the clearest choice.

## Final Pass

- Read the opening once more.
- Read the last sentence once more.
- Shorten any sentence that still feels heavy.
- If the text is a CTA, badge, or status line, confirm it still matches the UI state and is readable on the live theme.
